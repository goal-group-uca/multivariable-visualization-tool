## Requirements
The entire application is developed with Python 3. For its correct deployment it is necessary to install the following Python 3 libraries:
* branca
* contextily
* folium
* geopandas
* kivy
* kivymd
* matplotlib
* numpy
* pandas
* Requests
* scipy
* seaborn
* Shapely
## Deployment
` python3 appMapVisualizer.py `
